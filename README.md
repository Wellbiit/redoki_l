Проєкт Readoki – це сайт, що допоможе людям організовувати свій процес читання та бачити прогрес. Фактично він є читацьким щоденником, де користувач може вписувати прогрес читання та додавати помітки, коментарі щодо прочитаних книг, організовувати їх у папки і тд.
Даний проєкт створений з використанням HTML, CSS та JS файлів.
Користувач може авторизуватись на сайті, переглядати свою бібліотеку, відстежувати прогрес читання та додавати помітки.
Використані технології:
1. HTML5
2. CSS3

Тепер ви можете також користуватись сайтом і на мобільному телефоні.
Бо сайт тепер містить адаптивне меню та валідацію форм.

Додана JavaScript-функціональність:
1. DOM.
   Було додано анімацію та тінь до книжкових карток та в кінці кожного блоку <main> було додано новий абзац за допомогою createElement() та append().
2. Динамічна зміна контенту.
   У футері додано відображення поточної дати. Реалізовано кнопку "Показати більше" на сторінці "Про нас", що містить прихований текст (акордеон).
3. Робота з подіями.
   Додано перемикач зі світлої теми на темну, та навпаки. Також реалізовнавано підсвітку іконок меню при наведенні миші та зміна розміру шрифту за допомогою кнопок ArrowUp, ArrowDown.
4. Робота з формами та валідацією.
   Валідація полів форми відповідно з обмеженнями; відображення тексту помилки під полями; підсвічування полів, що не пройшли валідацію; виведення даних у консоль; в разі успішної валідації відбувається очищення форм та показ повідодмлення "Форма успішно надіслана!".
5. LocalStorage.
   Збереження теми сайту для всіх сторінок.
